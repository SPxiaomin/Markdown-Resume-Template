### 鹰角网络 (Hypergryph) — 平台工程  
**2024年4月 – 至今**

#### 礼包及抽奖平台 (系统负责人)

- **项目背景**: 公司级可复用平台，实现业务活动与奖品发放的解耦。
- **规模指标**:
  - 单次活动支持高达 **100 万用户**
  - 累计 **数千万条抽奖记录**
  - 流量峰值：**130K QPS (抽奖)** / **500K QPS (信息查询)**
- **核心设计**:
  - Redis 集群 (16 × 16GB)，单个活动采用 **1024 个虚拟分片** 以消除热点 Key。
  - 使用 Kafka 确保消息可靠交付；利用 Redis ZSet 追踪待处理状态并实现幂等重试。
  - MySQL 采用 **按年 + 12 分区** 的分表策略，严格控制单表规模。
- **成果**:
  - **10 分钟内处理 1000 万条 Kafka 消息并完成奖品发放**。
  - 显著降低了新活动的接入和开发成本。

#### 直发礼包与 CD-Key 系统

- 通过多分片 Redis 副本机制，支持约 **1M QPS 纯读流量**。
- 基于 RocketMQ 实现异步重试机制 (10分钟 / 6小时 / 12小时 / 24小时)。
- CD-Key 兑换基于 **纯数据库 + 按年 20 分片设计**，实现高吞吐量。
